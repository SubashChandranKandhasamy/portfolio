Projects

Architecture Design Hub
The Architecture Design Hub is an innovative web-based platform that connects users with professional architects to create customized building designs. The platform allows users to submit their land maps, explore pre-designed architecture templates, and contact architects for personalized modifications.
Online Blood Banking System
The Online Blood Banking System is a web-based platform designed to streamline blood donation and transfusion processes by connecting donors, recipients, and blood banks in an efficient and organized manner. The system ensures the availability of blood units based on demand, helping hospitals and patients find donors quickly.
Real-Time Chat System for Architect-Client Communication
The Real-Time Chat System is an interactive messaging platform that enables seamless communication between clients and architects. Designed to enhance collaboration, this system allows users to discuss architectural designs, share updates, and finalize building plans in real time.
